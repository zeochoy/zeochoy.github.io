About Me

Interested in the application of machine learning especially deep learning in oncology and development of bioinformatics tools for NGS data analysis.

I have worked on projects relating to microfluidics and mathematical modeling. And also have extensive wet lab experience, e.g. basic molecular biology techniques, tissue culture and drug testing.

Along with experts, doctors and scientist foresee the advancement in artificial intelligence could accelerate the progress of personalized and precision oncology. I have a strong belief that the combination of deep learning and next generation sequencing could be an indispensable catalyst, which fuelling my enthusiasm for applying deep learning on NGS data.

Projects

All Deep Learning Machine Learning Stochastic Modeling

DeepSAC

Deep learning based model for sequential application of chemotherapeutics, i.e. finding sensitizer for chemotherapy. Although trained on cell line data, we envision the model could help clinicians to identify the best treatment strategy for patients.

Gene Embedding

Using embedding to infer related genes from large molecular dataset. Embedding is able to represent categorical variables (in this case, genes) with continuous numbers, i.e. vector. Some interesting properties including gene relatedness and/or semantic meaning of genes are preserved and reflected in the embedding space.

Resim

Develop a python package with GUI powered by Flask to model cancers chemoresistance dynamics and predict chemoresistance onset.

Skinapp

Flask app to classify malignant skin lesion from benign moles using photo captured by mobile phone. Fine-tuned a pre-trained ResNext50 with HAM10000 dataset.
